# セキュリティ設計

## 1. パスワード管理

### 目的
ユーザーのパスワード漏洩を防ぐ

### ルール
- 平文保存禁止
- ハッシュ化して保存
- SHA256を使用

### 保存例

❌ NG

1234

⭕ OK

03ac674216f3e15c761ee1a5e255f067953623c8b388b4459e13f978d7c846f4

## 3. 権限管理

### 目的
他人のファイルアクセス防止

### ルール
- user_id を利用して所有者確認
- 所有者以外はアクセス禁止

### 確認項目
- ダウンロード
- 削除
- ファイル一覧

### 使用ファイル

security/permission.py
